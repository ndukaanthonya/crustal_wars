# Crustal Wars

**Play-to-Earn Arena Shooter on Celo**

A fast-paced mini-arena shooter where players pay a $2 cUSD entry fee, receive a code-generated sea creature NFT avatar, battle through 90-second matches, and earn $CRUST tokens.

## Tech Stack

- **Game Engine:** Phaser.js 3
- **Frontend:** HTML + CSS + JavaScript (Vite)
- **Smart Contracts:** Solidity 0.8.20 (Hardhat)
- **Chain:** Celo Alfajores Testnet
- **Backend:** Node.js + Express
- **Database:** Supabase
- **NFT Storage:** IPFS via Pinata

## Getting Started

```bash
npm install
npm run dev
npx hardhat compile
```
